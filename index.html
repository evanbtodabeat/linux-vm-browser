About This App

This is a browser-based Linux virtual machine emulator powered by v86, a JavaScript x86 PC emulator.

Features

  • ✅ Full Linux OS running in your browser
  • ✅ No installation or downloads required
  • ✅ Works offline (after initial load)
  • ✅ Multiple Linux distributions supported
  • ✅ File system persistence
  • ✅ Network access (with limitations)

How It Works

This app uses:

  • v86: JavaScript PC emulator
  • WebAssembly: For performance
  • Web Storage: For persistence

System Requirements

  • Modern browser (Chrome, Firefox, Safari, Edge)
  • At least 2GB RAM recommended
  • JavaScript enabled
